## What is Practice Call?
Practice Call is a website to set up quick, short timed exercises. [You can check it out here](https://practice-call.surge.sh)!

## The story behind Practice Call
I built this website to help me practice thumbnail and 3D sketching for set design. I wanted to create a site to help me quickly get ideas out of my head and onto a page. 

## Dev notes
**Features**
- Select the number of exercises
- Choose timing per exercise vs whole session
- Take a little break between exercises
- A rotation of public theatre images
